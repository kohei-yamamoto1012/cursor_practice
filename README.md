## サービス概要
「ショートカットキーを用いた、カーソルキーの移動」を練習するためのアプリケーションです。<br>
ショートカットキーをマスターすることで、ホームポジションを維持したまま作業を行えます！<br>

Javascriptの入門的な知識を学習し終えたタイミングで、学習内容の確認を兼ねて開発しました。<br>
開発期間: 3日

## 使用技術
- Ruby 2.6.8
- Rails 5.2.6
- Bootstrap4
